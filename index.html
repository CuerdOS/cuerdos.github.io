Acerca de

CuerdOS es una distribución de GNU/Linux enfocada en la estabilidad y rendimiento constante en ordenadores de cualquier gama, sin perder funcionalidad alguna.

Esta distribución no es solo una derivada con un entorno de escritorio personalizado, sino que tiene una serie de optimizaciones y mejoras de rendimiento, en los que se encuentran: la optimización de servicios activos, mejoras en el uso del procesador, entre otras muchas más.

Moviéndonos a Debian

Estamos trabajando para que la siguiente release de CuerdOS use la base de Debian 12.

Principales características

Otras características técnicas

  • Bauh Store preconfigurada para facilitar la instalación de programas (Debs y flatpak).
  • ZRam habilitado para mejor rendimiento en máquinas de bajos requisitos pero también para evitar escrituras en el SSD o tiempos de demora en el HDD por la SWAP.
  • Optimizado el uso de CPU y gestión de la memoria RAM.
  • TLP preconfigurado tanto para Intel como AMD para tener una mejor autonomía de la batería.
  • Pipewire como servidor multimedia.
  • Requisitos mínimos:
    • 1GB de ram mínimo (aunque se recomiendan 2GB).
    • Procesador AMD o Atom equivalente de 64 bits (no se especifica modelo del procesador Atom debido a que no hay apenas diferencias entre ellos en rendimiento).
    • Almacenamiento mínimo 15GB libres para no quedarnos cortos en caso de instalar más software (SSD recomendado para obtener la mejor experiencia, aunque con HDD también rinde bastante bien).
  • GRUB y boot logo personalizados junto al amado neofetch.
  • Polybar y Rofi (y Waybar y Wofi) preconfigurados y tematizados para conseguir una buena coherencia visual junto a los temas propios de iconos y ventanas.
  • Soporte a ser virtualizado.
  • Actualizaciones de seguridad automáticas.
  • Atajo para recordar otros atajos en Xfce, CTRL + ALT + H.
